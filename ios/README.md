## Build Locally

From this folder:

```sh
cd ios/source
pod install
open "Erva Mate.xcworkspace"
```

Then build/archive from Xcode.

## Signing

Do not commit Apple signing credentials.

Keep these outside git:

- Apple certificates.
- Provisioning profiles.
- App Store Connect API keys.
- Private keys.
- Passwords.
