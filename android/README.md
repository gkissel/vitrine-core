# Android Package

Android package generated by PWABuilder for the storefront PWA.

Production PWA URL:

```text
https://app.ervamateparaobrasil.com.br
```

Android package name from `assetlinks.json`:

```text
br.com.ervamateparaobrasil.app.twa
```

## Contents

- `releases/Erva Mate Para o Brasil.apk` — generated APK.
- `releases/Erva Mate Para o Brasil.aab` — generated Android App Bundle for Google Play.
- `source/` — generated Android/Gradle source project.
- `assetlinks.json` — Digital Asset Links file from PWABuilder.

The same `assetlinks.json` was also copied to:

```text
apps/front-end/public/.well-known/assetlinks.json
```

This lets Android verify that the app package is allowed to handle the production PWA domain.

## Build Locally

From this folder:

```sh
cd android/source
./gradlew assembleRelease
```

For Google Play, use the AAB generated by PWABuilder or build a new bundle from source:

```sh
cd android/source
./gradlew bundleRelease
```

## Signing

Do not commit signing credentials.

The original PWABuilder download included signing files, but keystores are intentionally excluded from the repository. Keep release signing credentials in a secure place outside git.

Ignored signing file examples:

- `*.keystore`
- `*.jks`
- `*.p12`
- `*.mobileprovision`
