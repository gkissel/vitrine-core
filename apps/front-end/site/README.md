# Storefront Site Overrides

Use this directory for site-specific storefront code that should not live in the base template's shared tree.

- `components/` for branded or site-specific UI
- `content/` for site-owned copy or content maps
- `lib/` for storefront-local helpers
- `routes/` for site-only route handlers or route support files
- `theme.css` for site-specific Tailwind v4 token overrides
